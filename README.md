# Number Guessing Game 🎮

A robust, interactive command-line Number Guessing Game built in Java. This project was developed as part of the **DecodeLabs Intern ** to demonstrate core Java programming concepts, input validation, and game logic design.

---

# Features

- **Dynamic Number Generation:** Generates a random target number between 1 and 100 for each round.
- **Interactive Feedback:** Provides real-time hints (`Too High!` or `Too Low!`) after every guess.
- **Attempt Limiter:** Restricts the player to **7 attempts** per round to increase difficulty.
- **Multi-Round Support:** Allows the player to play multiple consecutive rounds without restarting the application.
- **Score Tracking System:** Awards points dynamically based on how quickly the user guesses the correct number (fewer attempts used = higher score).
- **Robust Input Validation:** Prevents program crashes by safely handling non-integer inputs.

##  Prerequisites

Before running this project, ensure you have the following installed:
- **Java Development Kit (JDK):** Version 8 or higher.
- **Command Line/Terminal** (or an IDE like IntelliJ IDEA, Eclipse, or VS Code).
